# Futuristic Date Difference Calculator

This is a futuristic date difference calculator inspired by modern design principles. Enter two dates, and the tool will calculate the difference between them in days, months, and years. If the end date is earlier than the start date, it will display the difference as a negative value.

## Live Demo

You can use the calculator [here](https://urm1n.github.io/Date-Calculator/).

- **Negative Difference Display**: If the end date is earlier than the start date, it shows the difference as a negative value without any warning.
- **Responsive Design**: Optimized for use on all device sizes.

## Usage

1. Select a **Start Date** and an **End Date**.
2. Click **Calculate** to view the difference in days, months, and years.
3. The end date is set to today by default. If the end date is earlier than the start date, the difference will be shown in negative format.
